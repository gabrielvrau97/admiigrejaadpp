Pasta de imagens da marca (logo e cabeçalho)
=============================================

Coloque aqui os arquivos de imagem para usar nos relatórios impressos
e no cabeçalho do sistema.

Nomes esperados (use exatamente estes nomes para funcionar sem ajustes):

  logo.png           → logo principal (quadrada, ideal 512x512)
  logo-horizontal.png → logo horizontal (ideal 600x160, para cabeçalhos)
  cabecalho.png      → banner/cabeçalho completo do relatório (ideal 1600x300)

Formatos aceitos: .png, .jpg, .jpeg, .svg
Tamanho recomendado: até 500 KB por imagem

Como acessar no sistema:
  /brand/logo.png
  /brand/logo-horizontal.png
  /brand/cabecalho.png

Obs: por estarem em /public, essas imagens são servidas direto pelo Vite,
sem precisar de import no código.
